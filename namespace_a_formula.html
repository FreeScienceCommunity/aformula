static void AFormula::signal_handler ( int  signum  )  [static]

Trap for crash signals during backend testing.

On our speed test, it is possible that a given backend may simply crash entirely. We want to try to guard against that and insulate the user, so endeavor to catch any signals, raise a flag, and abort the testing of that module.
