Detailed Description

Abstract interface for code generator visitor.

We implement a visitor pattern (with double-dispatch) for just-in-time code compilation with any compiler. The parser creates a recursive-descent parsing tree, and the JIT formula classes implement this pattern, overriding the various emit() methods to produce appropriate code.

All of the emit() methods are defined to return void* pointers. These will in fact be whatever the appropriate class for the JIT compiler at issue is. For LLVMFormula class, this is an LLVM::Value* pointer, and for the LibJITFormula class, this is a jit_value_t pointer. The ponters are defined generically here to prevent the entire CodeGenerator, ExprAST, and Parser class structure from being template classes.
